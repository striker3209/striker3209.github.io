请编写一个JavaScript函数 parseQueryString，它的用途是把URL参数解析为一个对象

如：var url = ”http://www.taobao.com/index.php?key0=0&key1=1& key2=2”
var obj = new parseQueryString(url);
alert(obj.key0) // 输出0
